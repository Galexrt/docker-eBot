Docker eBot
==========

This is a Docker implementation of eBot.
Please be aware that this is only the server.

The web-version of this will be done soon.

Usage
-------
provide information for:

 - 	  eBot installation folder
 -    eBot IP-address
 -    MySQL-username
 -    MySQL-password
 -    MySQL-db

in the Dockerfile
